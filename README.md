# SIAKAD Learning Analytics

## Overview

SIAKAD Learning Analytics is a web-based Academic Information System developed as a final-year project. The system integrates Learning Analytics to help teachers monitor student academic performance through grade analysis and attendance tracking.

By combining academic management features with data analytics, the system provides meaningful insights into student learning outcomes and supports data-driven decision-making in the educational environment.

## Objectives

* Manage academic data efficiently.
* Record and process student grades.
* Monitor student attendance.
* Provide learning analytics dashboards for academic performance evaluation.
* Support teachers in identifying student achievement trends.

## Key Features

### Academic Management

* Student management
* Teacher management
* Subject management
* Class management
* Academic year management

### Assessment Management

* Learning objective (TP) management
* Assessment type management
* Student grade recording
* Automatic grade calculation

### Attendance Management

* Student attendance recording
* Attendance percentage calculation
* Attendance monitoring dashboard

### Learning Analytics Dashboard

* Student performance visualization
* Grade distribution analysis
* Attendance analysis
* Academic achievement monitoring

## Technology Stack

* Backend: Laravel
* Frontend: Blade, Bootstrap, JavaScript
* Database: MySQL
* Charts & Visualization: Chart.js
* Authentication: Laravel Authentication
